Key Features

Separate login & registration for farmers and buyers

Product management for farmers (add/update/delete products, set price)

Product catalog for buyers with category-wise items (crops, fruits, vegetables, etc.)

Shopping cart functionality and order placement

Order tracking for buyers

Persistent storage using MySQL with JDBC connectivity
